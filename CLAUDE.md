## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues (larced/pipe), using the `gh` CLI. External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context layout — `CONTEXT-MAP.md` at the root pointing to per-solution `CONTEXT.md` files under each library's directory. See `docs/agents/domain.md`.
